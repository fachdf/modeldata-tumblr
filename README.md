# modeldata-tumblr
How to run :
- 
1). Server :
use your terminal into /modeldata-tumblr/
- write this :
- - cd server
- - npm run server
- if there's an error do this :
- - npm install
- - redo the npm run server
- 2). Client :
open another terminal into /modeldata-tumblr/
- write this :
- - cd client
- - npm run serve
- if there's an error do this :
- - npm install vue
